JBang with Spring Boot
===================================
          

# JBang Templates
   
```
$ jbang trust add https://github.com/linux-china/jbang-spring-boot
```
      
### Spring Boot with Java 

* Spring Boot 2: `jbang init -t SpringBoot2@linux-china/jbang-spring-boot`
* Spring Boot 3: `jbang init -t SpringBoot3@linux-china/jbang-spring-boot`

### Spring Boot with Kotlin 

* Spring Boot 2: `jbang init -t SpringBoot2Kt@linux-china/jbang-spring-boot`
* Spring Boot 3: `jbang init -t SpringBoot3Kt@linux-china/jbang-spring-boot`
